Certificates are hosted on Google Drive (opens in a new tab, no LinkedIn login).

Setup (same as a typical portfolio):
1. Upload certificate image/PDF to Google Drive
2. Right-click → Share → General access: Anyone with the link
3. Copy link, e.g. https://drive.google.com/file/d/1Ngh0WERl-axXFbJmnD-XMffJNxOHOtJN/view?usp=sharing
4. Paste the file ID (the long string after /d/) into script.js → CERT_DRIVE_IDS

Example file ID from a share link: 1Ngh0WERl-axXFbJmnD-XMffJNxOHOtJN
